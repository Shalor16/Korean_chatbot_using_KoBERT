# Korean_chatbot_using_KoBERT
# KoBERT를 이용한 BERTScore 유사도를 이용한 챗봇

데이터셋으로 쓰인 소상공인 고객 주문 질의-응답 데이터셋은 한국지능 정보사회진흥원의 사업결과로서 AIHub를 통해 제공받음을 공시합니다.  
AIHub의 데이터 이용정책에 따라 데이터셋은 업로드하지 않으며, 대신 AIHub 링크를 남깁니다.
